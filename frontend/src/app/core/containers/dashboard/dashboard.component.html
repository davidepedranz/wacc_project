Dashboard

Welcome to our Docker Dashshboard.

You can do the following:

  • Services
    • Visualize the services running in the Docker Swarm.
    • Create new services.
    • Scale existing services.
    • Delete a service.
  • Events
    • Visualize the events occuring internally to Docker Swarm, in real time.
  • Users
    • Visualize the users in the system.
    • Create new users.
    • Add or remove permissions.
    • Delete users.
